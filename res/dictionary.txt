# This file contains localized text strings, each entry goes like this:
# <key>:<lang>:<value>
#
# where:
# - <key> is the key used by the application to designate this entry
# - <lang> is a 2-letter country code (EN, FR, JP...)
# - <value> is the localized text in the <lang> language
#
# IMPORTANT: This file should be saved in UTF-8 character encoding.
#
# Some things to keep in mind when localizing texts into a new language:
# - you should never modify keys, only their localized values
# - when translating an entry which has a variable (%1, %2...) placed at the end,
#  it might be because its value at runtime can be very large and thus will be
#  truncated where it appears. In this case, please try to use a sentence that
#  puts the variable at the end as well.
#
